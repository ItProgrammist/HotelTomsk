Данный проект создан для бронирования номеров отеля. На главной вкладке можно немного ознакомиться с нашим отелем и просмотреть действующие контакты для связи с администрацией отеля.

Для входа в свой личный кабинет необходимо сначала зарегистрироваться, потом войти. Личный кабинет представляет собой форму заполнения данных для бронирования номера. Также можно выбрать любой свободный номер для проживания (1-3 этаж). Повторно войдя в свой кабинет под своим логином (ФИО), можно поменять забронированный номер (у каждого пользователя свой номер, который он выбрал, и менять он может только свой, ранее забронированный номер). Имеется счетчик, благодаря которому можно узнать сколько всего номеров забронировано.

Для администратора есть специальная панель, включающая в себя некоторые функции: 1. Можно снять занятость любого из номеров или снять занятость всех номеров в отеле; 2. Можно сбросить всех зарегистрированных пользователей, снимая при этом занятость всех забронированных номеров отеля.
