
RUSSIAN LANGUAGE

  Данный проект создан для бронирования номеров отеля. На главной вкладке можно немного ознакомиться с нашим отелем и просмотреть действующие контакты для связи с администрацией отеля.

  Для входа в свой личный кабинет необходимо сначала зарегистрироваться, потом войти. Личный кабинет представляет собой форму заполнения данных для бронирования номера. Также можно выбрать любой свободный номер для проживания (1-3 этаж). Повторно войдя в свой кабинет под своим логином (ФИО), можно поменять забронированный номер (у каждого пользователя свой номер, который он выбрал, и менять он может только свой, ранее забронированный номер). Имеется счетчик, благодаря которому можно узнать сколько всего номеров забронировано.

  Для администратора есть специальная панель, включающая в себя некоторые функции: 1. Можно снять занятость любого из номеров или снять занятость всех номеров в отеле; 2. Можно сбросить всех зарегистрированных пользователей, снимая при этом занятость всех забронированных номеров отеля.

  P.S. Если хотите побыть в роли администратора, то: пароль от панели администратора: paneladmina
!!!ИЗМЕНЕНО/ДОБАВЛЕНО!!!: 1. По выбору номера (3м. "Люкс", например) открываются только номера, соответствующие указанным параметрам и счетчик ведется для каждой группы; 2. В панели администратора (в форме: "Снять занятость номеров") отображаются только те кнопки (номера), которые уже забронированы.
 
 
ENGLISH LANGUAGE
 
  This project was created for a hotel room. On the main tab, you can familiarize yourself a little with our hotel and view contacts for contacting the hotel administration.
 
  To enter your personal account, you must first register, then log in. Personal account is a form for filling in data for booking a room. You can also choose any free room for living (1-3 floor). By re-entering your account under your username (full name), you can change the booked room (each user has his own number, which he has chosen, and he can only change his previously booked number). There is a counter, thanks to which you can find out how many rooms are booked in total.
  
  There is a special panel for the administrator, which includes some functions: 1. You can remove the occupancy of any of the rooms or remove the occupancy of all rooms in the hotel; 2. You can reset all registered users, while removing the occupancy of all booked hotel rooms.
  
  P.S. If you want to be in the role of administrator, then: password from the administrator panel: paneladmina
!!! CHANGED / ADDED !!!: 1. By choosing a number (3m. "Lux", for example), only numbers corresponding to the specified parameters are opened and the counter is kept for each group; 2. In the admin panel (in the form: "Remove the occupancy of numbers"), only those buttons (numbers) that have already been booked are displayed.
